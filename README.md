# Bhashika H. Jayasinghe — Portfolio

🚀 Personal portfolio website built with **React + Vite + Tailwind CSS v4**. Showcasing projects, skills &amp; experience as a Software Developer from Sri Lanka.

## Tech Stack

- **React 19** + **Vite 7** — frontend framework and build tool
- **Tailwind CSS v4** (via `@tailwindcss/vite`) — utility-first styling
- **React Router DOM v7** — client-side routing
- **Lucide React** — icon library
- **Netlify Forms** — serverless contact form

## Pages

| Route | Description |
|-------|-------------|
| `/` | Hero section with intro, stats, and CTA |
| `/about` | Bio, info card, and career timeline |
| `/skills` | Skill bars by category and tech badges |
| `/projects` | Featured and other project showcase cards |
| `/contact` | Netlify Forms-powered contact form |

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

This project is configured for **Netlify** deployment. The `netlify.toml` includes a redirect rule so React Router works correctly with direct URL access.
